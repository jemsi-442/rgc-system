Legacy modules archived from the active RGC application surface.

- Keep the production RGC app focused on the current governance modules.
- Preserve older repo-era code for manual reference without leaving it in the active runtime surface.

- Unused legacy controllers
- Unused legacy models
- Old export classes
- Old Blade modules for members, income, requests, pastoral services, settings, quick entry, and member portal
- Old seeders not used by DatabaseSeeder

These files are intentionally not loaded by active routes or the current seeding flow.
